Laboratório de Programação 1

Neste trabalho final, tive que desenvolver o jogo mastermind. Para isso foi usada a linguagem C, com auxílio da biblioteca Allegro para fazer a parte gráfica. O jogo consiste em adivinhar as 4 cores sorteadas pelo computador, tendo indicativos de posições certas e erradas. Caso as 4 bolinhas laterais sejam pretas, o jogador acertou e ganhou o jogo. Ao contrário após as 9 tentativas o jogador perde se não acertar a sequência correta.

Laboratório de Programação 2

O trabalho final de Laboratório de programação 2, consistia em fazer um sistema de gerenciamento de atendimentos para empresas de saúde. Em que o usuário passaria por uma triagem para ver sua prioridade de atendimento. Após isso, o funcionário realizava o atendimento, e processava os documentos que eram necessários (receitas médicas e atestados). E para finalizar o usuário poderá visualizar e imprimir esses documentos. Esse sistema foi todo feito com pilhas e filas e árvores dinâmicas na linguagem C.

Estrutura de Dados

O trabalho final de Estrutura de Dados consistia em fazer um sistema de cadastro de veículos. Que tem as funcionalidades de cadastrar, listar, remover e buscar veículos. O sistema foi feito na linguagem C, utilizando árvores binárias de busca e o próprio prompt da linguagem.
